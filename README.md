# confighub

一键搭建个人生产力环境。

## 快速开始

1. 获取本项目，并替换`~/.config`目录。

  - 短期使用：直接下载替换。

  - 长期使用：

    ````sh
    git clone https://github.com/afzw/confighub.git ~/.config
    ````


2. 执行`./build.sh`脚本，使配置生效。

3. 参考本项目中的说明文档，安装相应的软件，调试配置。

## 目录说明

> **不带有`_`前缀的目录基本都是相关软件的配置。**

- `_bin`目录存放自定义命令。
- `_docs`目录存放说明文档
- `_manual`目录存放需要手动维护的配置。
